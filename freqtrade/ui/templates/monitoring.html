🤖 DSPy Machine Learning Advisor

DSPy is a machine learning platform that observes trading performance and generates intelligent parameter optimization suggestions using bounded control guardrails.

How it works: The DSPy ML platform continuously analyzes metrics like Sharpe ratio, drawdown contribution, and capital efficiency to identify optimization opportunities. All suggestions are safety-bounded (±10% for allocation weights, ±20% for thresholds) and require human approval before application.
Key principle: OBSERVE → ANALYZE → SUGGEST → LOG (NEVER AUTO-APPLY). Safety overrides intelligence.
